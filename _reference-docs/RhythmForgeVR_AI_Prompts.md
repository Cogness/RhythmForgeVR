## PART 1: ELEVENLABS VOICEOVER PROMPTS

### Setup in ElevenLabs

1. Go to [elevenlabs.io](https://elevenlabs.io) → **Speech** (Text to Speech).
2. **Voice selection:** Choose one of these free voices (sorted by best fit):
   - **"Brian"** — calm, confident, male. Ideal for tech pitch.
   - **"Chris"** — warm, natural male voice. Good alternative.
   - **"Aria"** — clear, professional female voice. Works well if you prefer a female narrator.
3. **Settings:**
   - Stability: **0.55** (slightly expressive, not robotic)
   - Similarity: **0.80** (keeps the voice consistent)
   - Style: **0.30** (subtle expressiveness)
   - Speaker Boost: **ON**
4. **Model:** Select **"Eleven Multilingual v2"** or **"Eleven Turbo v2.5"** for best quality.

### Generate Each Beat Separately

Generate each beat as a separate audio file. This gives you precise control in the editor. Name the files clearly (`beat1_voice.mp3`, `beat2_voice.mp3`, etc.).

## PART 2: GOOGLE VEO 3.1 VIDEO PROMPTS

### How to Use Google Veo 3.1

1. Go to [Google AI Studio](https://aistudio.google.com/) or [VideoFX](https://labs.google/fx/tools/video-fx) (both provide Veo access).
2. Enter the text prompt.
3. Set duration to **4 seconds** (this is Veo's standard generation length — you will trim to 2–3 seconds in your editor).
4. Generate 3–4 variations per prompt. Pick the best one.
5. Download as MP4.

### General Tips for Better Veo Results

- Be specific about camera angle, lighting, and movement.
- Include "cinematic" or "shallow depth of field" for professional-looking results.
- Mention what should NOT be in the frame if needed.
- Keep prompts under ~200 words for best results.

## PART 3: END CARD

No AI generation needed. Create this in any image tool.

### Option A: Create in Canva (free)

1. Go to [canva.com](https://canva.com) → Create Design → Custom Size → 1920 × 1080 px.
2. Background: solid **#0A0A0A** (near-black).
3. Add text, centered:

```
RhythmForge VR

Conduct reality. Compose destiny.
Create music with motion.

Built for Logitech MX Ink × Meta Quest
DevStudio 2026
```

4. Font suggestions: **Inter**, **Space Grotesk**, or **DM Sans** (all free on Canva).
   - "RhythmForge VR" — 72pt, bold, white (#FFFFFF)
   - Tagline — 28pt, regular, light gray (#AAAAAA)
   - "Built for..." line — 20pt, regular, medium gray (#777777)
5. Export as PNG.
6. Import into your video editor as the last frame. Display for **3 seconds**.

### Option B: Create in Keynote (free, on Mac)

1. New presentation → Black slide.
2. Add text boxes with the same content and sizing as above.
3. File → Export → Images → PNG → export just that slide.
