# RhythmForge Audio Direction

## When To Use

Use this skill when the task is about what the app should sound like, how presets should be structured, or how audio events and mix priorities should be organized.

## Primary Reference

Read `docs/ai/audio.md` first.

## RhythmForge Rules

- Treat audio as product-defining, not secondary polish.
- Prefer production-oriented samples and presets over toy-like synthesis.
- Keep event ownership and preset slot roles explicit.
