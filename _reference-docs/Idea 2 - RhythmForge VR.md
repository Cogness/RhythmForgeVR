## 🎨 FEATURE BREAKDOWN (MVP for Hackathon)

### **1. Rhythm Canvas (Stylus Primary Mode)**

**How It Works:**
- Draw circular or linear paths in space with MX Ink
- Path automatically loops
- Each time loop crosses a "trigger line", sound plays
- Pressure sensitivity = velocity/volume of hit

**Visual:** Think **SoundBow** meets **Tilt Brush** - your strokes glow and pulse with rhythm[6]

**Example:**
```
User draws circle in air → Circle loops infinitely
Places 4 "trigger points" on circle → Kick, snare, hi-hat, clap
Circle rotates = 4/4 beat pattern emerges
Pressure on each stroke = loudness of each hit
```

### **2. Melody Sculptor (Stylus Secondary Mode)**

**How It Works:**
- Switch mode via MX Ink button press
- Draw curves in space (height = pitch, length = duration)
- Curve becomes playable melody that loops
- Tilt stylus to bend pitch (whammy bar effect)

**Visual:** Musical staff appears, your drawing snaps to scale notes

**Example:**
```
Draw ascending curve → Creates melody "do-re-mi-fa-sol"
Loop melody → Plays continuously
Add pressure variation → Vibrato on certain notes
Tilt stylus → Pitch bends like guitar string
```

### **3. Instrument Library (Left Hand)**

**UI:** Floating radial menu anchored to left hand

**Options:**
- **Drums:** Kick, snare, hi-hat, toms, cymbals
- **Bass:** 808 sub, synth bass, acoustic bass
- **Melody:** Piano, strings, synth leads, pads
- **FX:** Risers, impacts, transitions

**Interaction:**
- Point at instrument → Spawns as 3D object
- Throw into rhythm loop → Plays with that pattern
- Scale object = volume control

### **4. Spatial Audio Mixing (Both Hands)**

**The Innovation:**
- Every sound is a 3D object floating around you
- **Closer objects = louder**
- **Left/right position = stereo panning**
- **Height = filter frequency** (high = brighter, low = darker)
- Grab with left hand, paint effects with stylus

**This is revolutionary** - mixing becomes **spatial sculpture** rather than fader adjustment.

### **5. Timeline & Export (Left Hand Gesture)**

- Swipe left hand down → Summons timeline below
- Pinch and drag to scrub through composition
- Record mode: Capture 1-4 minute loops
- Export as WAV/MP3 or share to SoundCloud
